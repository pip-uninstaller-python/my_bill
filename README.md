# 技术点

-   基于Flutter2.+，支持空安全
-   GetX
-   清晰的项目结构 (参考[getx_pattern](https://kauemurakami.github.io/getx_pattern/)规范)
-   Dio网络请求
-   屏幕适配
-   统一缺省页
-   统一刷新组件
-   视频播放(fijk)
-   弹幕播放
-   瀑布流
-   lottie动画
-   统一启动页
-   适配web,ios,macos(**win暂未适配**)
-   ...
